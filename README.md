# О чем проект #
Проект представляет собой калькулятор для расчета парметров трехкорпусной выпарной установки.

# Как в нем работать? #
Пользователь вводит параметры на главной странице. Он может создать свои растворы и добавить их для расчета, может выбрать уже существующие в меню. 
Введенные значения можно удалить или сохранить в css, который потом можноьимпортировать в программу. 
На дополнительной вкладке «Методика расчета» можно просмотреть дополнительную информацию о трехкорпусных выпарных установках. 
На странице «Результаты» можно получить все рассчитанные значения, сохранить и посмотреть графики, экспортировать результаты в css.

# Стэк технолгий #
Язык программирования: Python 3.12.6 Flask.
Язык разметки: HTML/CSS.
Язык скриптов: JS.

# Как запустить? #
1. Клонируйте репозиторий:
git clone https://github.com/alextochanka/calculator
cd calculator
2 Создайте и активируйте виртуальное окружение:
python -m venv venv
source venv/bin/activate
3 Установите необходимые библиотеки:
pip install -r requirements.txt

# Структура проекта #
1. templates - папка с HTML.
2. calculations.py - скрипт расчетов.
3. app.py - скрипт оформления калькулятора.
4. main.py - скрипт запуска калькулятора.
5. static - папка с CSS/JS и схемой установки.
6. render.yaml - скрипт запуска на Render для публикация приложения.
7. requirements.txt - список зависимостей.
