# О Чем проект #
Проект представляет собой калькулятор для расчета парметров трехкорпусной выпарной установки.

# Как в нем работать? #
Пользователь вводит параметры на главной странице. Он может создать свои растворы и добавить их для расчета, может выбрать уже существующие в меню. 
Введенные значения можно сохранить в css, который потом можно будет импортировать в программу. 
На дополнительной вкладке «Методика расчета» можно просмотреть дополнительную информацию о трехкорпусных выпарных установках. На странице «Результаты»
можно получить все рассчитанные значения, сохранить и посмотреть графики и экспортировать результаты в css.

# Стэк технолгий #
Язык программирования: Python 3.12.6 Flask.
Язык разметки: HTML/CSS.
Язык скриптов: JS.
